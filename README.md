# Toduo

Приватный чат-портал с задачами в стиле Notion — для команды из двух человек.
Место, где рождаются задачи (ТЗ → процесс работы → результат с вложениями) и где
можно переписываться в реальном времени.

## Что уже готово (этап 1)

- **Авторизация** — регистрация и вход, сессия на httpOnly-cookie.
- **Оболочка приложения** — сайдбар (Чат / Задачи), шапка с профилем и выходом.
- **Живой чат** — сообщения в реальном времени через SSE, история хранится в SQLite.
- **Задел под задачи** — раздел «Задачи» (заглушка), готовый принять модуль.

## Стек

- [Next.js](https://nextjs.org/) (App Router, TypeScript) — fullstack.
- [Tailwind CSS](https://tailwindcss.com/) — оформление.
- [better-sqlite3](https://github.com/WiseLibs/better-sqlite3) — база данных (SQLite).
- Realtime — Server-Sent Events (SSE) + внутренний in-memory pub/sub.
- Пароли — `bcryptjs`.

## Запуск

```bash
npm install
npm run dev
```

Откройте <http://localhost:3000>. При первом запуске зарегистрируйте аккаунты
(каждому из двоих — свой). База и загрузки создаются локально.

## Проверка живого чата

1. Откройте приложение в двух разных браузерах/профилях и войдите под разными
   аккаунтами.
2. Отправьте сообщение из одного окна — во втором оно появится сразу, без
   перезагрузки.
3. Обновите страницу — история подтянется из базы.

## Где лежат данные

- `data/toduo.db` — база SQLite (создаётся автоматически, в `.gitignore`).
- `uploads/` — папка под будущие вложения задач (в `.gitignore`).

Обе директории не коммитятся.

## Полезные команды

```bash
npm run dev     # разработка
npm run build   # сборка + проверка типов
npm run start   # запуск собранной версии
npm run lint    # линтер
```
